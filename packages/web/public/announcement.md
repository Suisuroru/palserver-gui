3. 記得:安裝、更新、移除模組都要**先停止伺服器**。

可以追蹤 PalDefender / UE4SS 的官方發布頁,或到我們的 [Discord](https://discord.gg/sgMMdUZd3V) 問一聲,有相容版我們也會在這裡公告。

Heads-up, island owners~

Palworld just updated to **1.0**, a big one. Mods that hook into the game — like **Palguard (now PalDefender)** and **UE4SS** — usually **stop working temporarily** after a major update, until their authors ship a 1.0-compatible version (typically within a few days).

**This is not a palserver GUI problem** — it's the same for every server: when the game changes, mods have to catch up.

## If your server won't start or crashes after updating to 1.0

It's most likely old, incompatible mods. Try this first:

1. Go to the **Mods** tab and **remove PalDefender / UE4SS** (the new version has a "remove" button) so the server can start normally.
2. In a few days, once the authors update, come back to the **Mods** tab and hit "**update to the latest version**" to reinstall.
3. Remember: installing, updating or removing mods all require **stopping the server first**.

## Want to know when it'll work?

Follow PalDefender / UE4SS's official release pages, or ask on our [Discord](https://discord.gg/sgMMdUZd3V) — we'll announce compatible versions here too.

Hang in there through the update growing pains, and enjoy 1.0! 🐾

1. **MOD** タブで **PalDefender / UE4SS を削除**(新版には「削除」ボタンがあります)し、サーバーを正常に起動できるようにします。
2. 数日後、作者が更新したら **MOD** タブに戻り「**最新版に更新**」で入れ直します。
3. Mod の導入・更新・削除は、必ず**先にサーバーを停止**してから。

## いつ使えるようになる?

PalDefender / UE4SS の公式リリースページを確認するか、[Discord](https://discord.gg/sgMMdUZd3V) で聞いてください。互換版が出たらここでもお知らせします。
